# 🍳 Магазин Кастрюль - Practice 5

## Описание
Практическое занятие №5 по дисциплине "Фронтенд и бэкенд разработка"

## Тема
Расширенный REST API с Swagger документацией

## Технологии
- **Backend:** Node.js + Express + Swagger (swagger-jsdoc, swagger-ui-express)
- **Frontend:** React + Axios + SASS

## Функционал
- ✅ CRUD операции для товаров
- ✅ Swagger документация API
- ✅ Интерактивный режим (Try it out)
- ✅ JSDoc-аннотации для всех эндпоинтов

## Запуск

### Backend:
```bash
cd backend
npm install
node app.js
```

### Frontend:
```bash
cd frontend
npm install
npm start
```
